# rRNA Modification Visualization

UCSC Genome Browser assembly hub for visualizing RNA modifications on human rRNA sequences (18S, 28S, 5.8S, 5S), plus whole-genome variant calls on hg38.

## How to run the pipeline

### Requirements

- Python 3.7+
- UCSC command line utilities for macOS Apple Silicon:

```bash
curl -O https://hgdownload.soe.ucsc.edu/admin/exe/macOSX.arm64/bedToBigBed
curl -O https://hgdownload.soe.ucsc.edu/admin/exe/macOSX.arm64/faToTwoBit
chmod +x bedToBigBed faToTwoBit
```

Place both tools in the same directory as `pipeline.py`, or anywhere on your PATH.

### Steps

**1. Run the pipeline from the repo root:**

```bash
cd /Users/baihesun/Downloads/rRNA_dataviz_repo
python scripts/pipeline.py
```

This will:
- Recolor each BED file using fixed per-modification-type colors with saturation scaled by frequency
- Rescale scores to 0–1000 range (required by UCSC)
- Keep all 11 columns including coverage and frequency as extra fields
- Scan the data to determine the modification types present and populate track filters
- Generate an AutoSql `.as` file (`rRNA_mods.as`) describing all 11 columns
- Convert each BED file to bigBed format using `bedToBigBed -type=bed9+2 -as=rRNA_mods.as`
- Generate `hub.txt`, `genomes.txt`, `trackDb.txt`, per-track HTML files, and `hubDescription.html` in `ucsc_hub/`

**2. Convert the FASTA reference to .2bit format (only needed once):**

```bash
./faToTwoBit Test_visualization_data_rRNA/hs_rRNAs_NR_046235.fa ucsc_hub/hs_rRNAs_NR_046235.2bit
```

**3. Push everything to GitHub:**

```bash
git add .
git commit -m "Update hub files"
git push -u origin main
```

## How to load the hub in UCSC

1. Go to [genome.ucsc.edu](https://genome.ucsc.edu)
2. Click **My Data** → **Track Hubs** → **My Hubs**
3. Paste the following URL and click **Add Hub**:

```
https://baihesun.github.io/rRNA_dataviz/ucsc_hub/hub.txt
```

## Tracks

### hs_rRNA (custom genome)

| Track | Description | Filters |
|---|---|---|
| Reference mods | Curated known modifications from published literature | — |
| MRI01 detected | Modifications detected in sample MRI01 | Frequency range, modification type |
| Filtered mods | High-confidence filtered sites | Frequency range, modification type |
| Consensus mods | Union of MRI01 and filtered; color = observation frequency | — |

### hg38

| Track | Description |
|---|---|
| SRS000090 variants | WGS merged variant calls (two-caller) streamed from GCS |

The hg38 VCF track is hosted externally at `gs://srs000090-wgs/` and is not stored in this repo.

## BED columns

Each rRNA track retains all 11 columns from the original bedRmod format. Clicking a site in UCSC shows all fields in a popup.

| Column | Field | Description |
|---|---|---|
| 1 | chrom | rRNA molecule (e.g. hs_rRNA_18S) |
| 2–3 | chromStart / chromEnd | Position of the modified site (0-based) |
| 4 | name | Modification type (e.g. Am, Y, m6A) |
| 5 | score | Detection score rescaled to 0–1000 |
| 6 | strand | Always + for rRNA |
| 7–8 | thickStart / thickEnd | Same as start/end (display only) |
| 9 | itemRgb | Color encoding mod type and frequency (see below) |
| 10 | coverage | Number of reads covering this position |
| 11 | frequency | Percentage of reads showing the modification (0–100) |
