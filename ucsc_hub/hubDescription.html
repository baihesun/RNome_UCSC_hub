Human rRNA Modifications

RNA modification tracks for human rRNA sequences (18S, 28S, 5.8S, 5S).

Each site is colored by modification type. Color saturation encodes stoichiometry: vivid = highly modified site, pale/gray = partially modified site.

Score

The score reflects detection confidence, rescaled to 0–1000 per file. Higher score = higher confidence. Scores are rescaled independently per file and are not directly comparable across tracks.

Filters (MRI01 detected and Filtered mods tracks)

Two interactive filters are available on these tracks via the track settings panel:

  • Modification frequency (%) — range slider to show only sites above a minimum frequency
  • Modification type — checklist to show only selected modification types

Consensus track

The Consensus track merges the sample and filtered tracks. Color encodes how often the modification is observed across inputs:

ObservationColor
1 / 2 samplesYellow
2 / 2 samplesGreen

Extra fields (visible on click)

Clicking any site shows coverage (raw read depth) and frequency (% of reads showing the modification).
